# 科學計算 -- 使用 python

套件安裝

```
$ pip install numpy
$ pip install matplotlib
$ pip install scipy
$ pip install panda
$ pip install sympy
$ pip install opencv-contrib-python // 之後使用 import cv2 (精簡版 pip install opencv-python)
$ pip install opencv-contrib-python-headless // opencv 的視窗介面 (精簡版 pip install opencv-python-headless)
```
